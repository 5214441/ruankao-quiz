柱子软考智能备考 V3.5 部署步骤

一、先不要删除当前网站
1. 下载并解压“柱子软考智能备考V3.5_GitHubPages升级包.zip”。
2. 在GitHub仓库里先保留当前版本，便于出问题时恢复。

二、上传新版全部文件
1. 进入仓库 5214441/ruankao-quiz。
2. 点击“Add file”。
3. 点击“Upload files”。
4. 将解压后的全部文件和文件夹拖入上传区域。
5. 页面提示同名文件时选择覆盖。
6. 点击“Commit changes”。

三、设置GitHub Pages部署方式
1. 打开仓库“Settings”。
2. 左侧点击“Pages”。
3. 找到“Build and deployment”。
4. Source选择“GitHub Actions”。

四、等待自动部署
1. 返回仓库首页。
2. 点击顶部“Actions”。
3. 打开“校验题库并部署 GitHub Pages”。
4. 等待绿色对勾。
5. 打开原网站地址：https://5214441.github.io/ruankao-quiz/

五、以后增加题库
1. 先用网站中的“设置 → 题库管理工具”校验候选题。
2. 把审核通过的JSON上传到 incoming 文件夹。
3. 有效新题累计达到10道后，GitHub Actions自动合并并更新网站。
4. 也可以在Actions页面手动运行更新任务。

重要提醒
- 新版已经拆分文件，不能在电脑上直接双击index.html预览。
- 上传时必须同时上传data、css、js等文件夹，否则网页会显示题库加载失败。
- 学习记录仍保存在浏览器本机，升级前建议在旧版“设置”中导出一次学习记录。
